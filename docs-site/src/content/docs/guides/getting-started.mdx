## Create a New Project

```bash
npx create-then my-app
cd my-app
npm install
npx thenjs dev
```

Your app is running at `http://localhost:3000`.

## Project Structure

```
my-app/
  src/
    pages/
      index.tsx          # → /
      _layout.tsx        # Root layout wrapper
    api/
      health.ts          # → GET /api/health
    entry-client.tsx     # Client hydration
    entry-server.tsx     # SSR render
  then.config.ts         # Configuration
  tsconfig.json
  package.json
```

## Your First Page

Pages live in `src/pages/`. Export a default component and an optional `page` config:

```tsx
// src/pages/index.tsx
export const page = { mode: 'hybrid' };

export default function Home() {
  return (
    <main>
      <h1>Welcome to ThenJS</h1>
      <p>The meta-framework for What Framework.</p>
    </main>
  );
}
```

Page modes: `'client'` (SPA), `'server'` (SSR every request), `'static'` (pre-rendered at build), `'hybrid'` (SSR + client hydration).

## Your First API Route

API routes live in `src/api/`. Export named functions matching HTTP methods:

```ts
// src/api/health.ts
export function GET() {
  return Response.json({
    status: 'ok',
    timestamp: new Date().toISOString(),
  });
}
```

## Your First RPC Procedure

For the full-stack template (`npx create-then my-app --template full-stack`), RPC procedures live in `src/rpc/`:

```ts
// src/rpc/user.ts
import { procedure, router } from '@thenjs/rpc';
import { z } from 'zod';

export const userRouter = router({
  getById: procedure
    .input(z.object({ id: z.string() }))
    .query(async ({ input }) => {
      return { id: input.id, name: 'Ada Lovelace' };
    }),

  create: procedure
    .input(z.object({ name: z.string(), email: z.string().email() }))
    .mutation(async ({ input }) => {
      return { id: crypto.randomUUID(), ...input };
    }),
});
```

Call from the client with full type safety:

```ts
import { createRPCClient } from '@thenjs/rpc';
import type { AppRouter } from './rpc/index.js';

const rpc = createRPCClient<AppRouter>();

const user = await rpc.user.getById.query({ id: '1' });
```
